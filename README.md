# Off-Road Command

Raspberry Pi-based passenger-side vehicle display for off-road vehicles.

## Planned features

- Offline navigation
- GPS routes and track recording
- GPX and KML import/export
- Persistent waypoints
- Bluetooth phone audio and media control
- VHF radio control
- Intercom integration
- MEFI-4 engine data over CAN
- Vehicle controls and solid-state PDM integration
- Camera support

## Prototype hardware

- Raspberry Pi 4 Model B, 4 GB
- Raspberry Pi OS 64-bit
- Qt 6
- PySide6
- Qt Quick/QML
- Qt WebEngine
- Leaflet

## Project status

Current prototype includes:

- Full-screen modular touchscreen interface
- Interactive map
- Simulated vehicle location
- Map pan and zoom
- Navigation and vehicle drawers
- Waypoint database backend
- Initial Bluetooth audio development
